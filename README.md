# Amazon-homepage-clone
A responsive frontend clone of Amazon's homepage and sign-in page built using HTML, CSS, and JavaScript. This project was created to strengthen my understanding of responsive layouts, UI design, and frontend development by recreating the interface of a real-world e-commerce website.

## Features

- Responsive homepage layout
- Amazon-inspired navigation bar
- Product categories and promotional sections
- Search bar interface
- Sign-in page
- Responsive footer
- Hover effects and smooth UI interactions

## Tech Stack

- HTML5
- CSS3
- JavaScript
